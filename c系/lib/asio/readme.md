# asio

Asio 是一个跨平台的 C++ 库，常用于网络编程、底层的 I/O 编程等 (low-level I/O)。

Asio 库分为 Boost 版和 non-Boost 版。

- **non-Boost版本官方网站** —— **[http://think-async.com/](http://think-async.com/)**
- **安装**

    ```bash
    ./configure --prefix=/usr/local/
    make -j4
    make
    ```
