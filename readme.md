# 智能合约虚拟机存在的意义

有3个节点，分布运行在不同的架构的环境中，最后运算的得出的结果很有可能是不一样的。对链来说，哪个计算的结果是正确的呢。此时就需要虚拟机来统一运算过程，最终得到一致的结果。这就是虚拟机存在的意义。
